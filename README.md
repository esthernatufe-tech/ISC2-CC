# ISC2 CC — Certified in Cybersecurity Practice Exam

An interactive, self-contained study tool for the **ISC2 Certified in Cybersecurity (CC)** certification exam.

Developed by **Esther Tosin Natufe** — built while studying for and passing the ISC2 CC exam. ✅

## 🔗 Live App

**[Open the study guide](https://esthernatufe-tech.github.io/ISC2-CC/)**

No installation, no sign-up, no app required — just click and study, on any phone, tablet, or laptop.

## What's inside

- **100 practice questions**, distributed across all 5 official CC domains using their real exam weightings:
  | Domain | Questions | Weight |
  |---|---|---|
  | Security Principles | 26 | 26% |
  | Business Continuity, Disaster Recovery & Incident Response | 13 | 13% |
  | Access Controls Concepts | 22 | 22% |
  | Network Security | 24 | 24% |
  | Security Operations | 15 | 15% |

- **Study notes** for every topic in each domain, written in plain language
- **Domain-by-domain practice mode** with instant feedback and explanations
- **Full Mock Exam** — 100 randomized questions with a live countdown timer paced like the real exam (~72 sec/question, ~2 hours total)
- **Quick Mock Exam** — a shorter 20-question version for fast practice sessions
- **AI-style question generator** — get extra practice questions per domain beyond the core 100
- **Progress tracking** for the current session (resets on page reload — no data is stored or sent anywhere)

## How to use it

1. Open the [live link](https://esthernatufe-tech.github.io/ISC2-CC/)
2. Pick a domain to review the study notes, or jump straight into practice questions
3. When you're ready, take the Full Mock Exam to simulate real exam conditions
4. Review your results — each question shows the correct answer and a full explanation

## Notes

- This is an independent study aid and is **not affiliated with or endorsed by ISC2**.
- Questions are written to match the style, difficulty, and topic coverage of the real exam, but are original — not leaked or official exam content.
- Everything runs entirely in your browser. No data is collected, stored, or transmitted.
